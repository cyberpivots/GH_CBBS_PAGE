# Prints

Future print-output workspace for prototype exports, slicer profiles, and print logs.

Suggested layout:
- `3mf/`: slicer project files.
- `stl/`: printable mesh snapshots copied from `cad/stl/`.
- `profiles/`: printer/material/slicer settings.
- `logs/`: print result notes, defects, photos, and fit feedback.
- `coupons/`: small calibration parts for inserts, snap fits, screw bosses, and clearances.

Minimum print log fields:
- Date
- Part revision
- Printer and slicer
- Material
- Nozzle size
- Layer height
- Orientation
- Supports
- Fit result
- Required CAD changes
