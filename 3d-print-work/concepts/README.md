# Concepts

Store early enclosure concepts here. Keep each concept in its own subfolder when work begins.

Recommended concept package:
- `brief.md`: target hardware, use case, constraints, and non-goals.
- `sketches/`: public-safe sketches or screenshots.
- `risks.md`: thermal, RF, environmental, assembly, and service risks.
- `decision-log.md`: concept decisions and evidence.

Initial concept candidates:
- `node-case`: Heltec node case with OLED window and antenna relief.
- `hub-radio-case`: Raspberry Pi hub/radio-head accessory case.
- `p070-panel`: P070 display bezel and mounting enclosure.
- `gateway-case`: future agriculture gateway or sensor case after hardware selection.
- `test-jigs`: programming, display alignment, and assembly fixtures.
