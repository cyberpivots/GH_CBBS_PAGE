# Validation

Use this folder for enclosure test plans and results.

Validation categories:
- `fit`: board, connector, antenna, display, and cable clearance.
- `assembly`: fastener engagement, insert retention, service access, repeated open/close checks.
- `thermal`: temperature checks under realistic workload.
- `rf`: LoRa/Wi-Fi smoke checks with final antenna routing and case material.
- `field-handling`: drops, vibration, mounting stress, and transport handling.
- `labeling`: public-safe labels, warnings, version marks, and access instructions.

Do not claim environmental ratings, thermal performance, or RF performance until tests are documented here.
