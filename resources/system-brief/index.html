What CBBS is

CBBS, also written as ClaRTK BBS, is an off-grid bulletin board and field-operations mesh. It is designed around durable local records, operator review, field node status, nearby browser access, and radio-backed communication paths.

Core layers

The system map highlights a Raspberry Pi hub, Heltec WiFi LoRa32 V2 field nodes, Windows operator tooling, local room-client browser access, and display surfaces for quick field visibility.

Current focus

CBBS focuses on communications, records, diagnostics, agriculture review, operator workflows, and realistic scenarios for rural and off-grid teams.
